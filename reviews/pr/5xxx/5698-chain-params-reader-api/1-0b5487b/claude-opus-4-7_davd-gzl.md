## Summary
Exposes the six `Get*` readers (`GetString`, `GetBool`, `GetInt64`, `GetUint64`, `GetBytes`, `GetStrings`) of `ParamsInterface` to Gno realms via `chain/params`, completing the round-trip with the existing `Set*` writers. The Get methods already live in [`execctx.ParamsInterface`](https://github.com/gnolang/gno/blob/0b5487b/gnovm/stdlibs/internal/execctx/context.go#L18-L36) · [↗](../../../../../.worktrees/gno-review-5698/gnovm/stdlibs/internal/execctx/context.go#L18-L36) and in [`SDKParams`](https://github.com/gnolang/gno/blob/0b5487b/gno.land/pkg/sdk/vm/builtins.go#L135-L157) · [↗](../../../../../.worktrees/gno-review-5698/gno.land/pkg/sdk/vm/builtins.go#L135-L157), so this PR is purely a Gno-side surface and a calibration entry per native. Key derivation reuses [`pkey()`](https://github.com/gnolang/gno/blob/0b5487b/gnovm/stdlibs/chain/params/params.go#L51-L60) · [↗](../../../../../.worktrees/gno-review-5698/gnovm/stdlibs/chain/params/params.go#L51-L60) (same validation as Set*), so a realm reads back exactly what it wrote.

## Glossary
- `pkey`: `"vm:" + rlmPath + ":" + key`, rejects `:` in `key`.
- `ParamsInterface`: Go-side contract bridging the VM to either the SDK keeper or the test stub.
- `TestAppHashCrossrealm38`: forward-guard test pinning the multistore root after a fixed scenario; any change to stdlib bytes shifts it.

  <details><summary>details</summary>

  The PR adds 13 Get* benchmarks in [`native_machine_bench_test.go:405-505`](https://github.com/gnolang/gno/blob/0b5487b/gnovm/cmd/calibrate/native_machine_bench_test.go#L405) · [↗](../../../../../.worktrees/gno-review-5698/gnovm/cmd/calibrate/native_machine_bench_test.go#L405) and 5 calibrator-table entries in [`gen_native_table.py:93-104`](https://github.com/gnolang/gno/blob/0b5487b/gnovm/cmd/calibrate/gen_native_table.py#L93) · [↗](../../../../../.worktrees/gno-review-5698/gnovm/cmd/calibrate/gen_native_table.py#L93). ## Nits

- [`gnovm/stdlibs/chain/params/params.gno:13-22`](https://github.com/gnolang/gno/blob/0b5487b/gnovm/stdlibs/chain/params/params.gno#L13) · [↗](../../../../../.worktrees/gno-review-5698/gnovm/stdlibs/chain/params/params.gno#L13) — the doc comment ahead of `GetString` reads "GetXxx reads back a value previously stored under key with the matching SetXxx" — worth one more sentence on what happens if the same key was set with a different type (e.g. `SetBool` then `GetString`). - [`gnovm/stdlibs/chain/params/params.go:53-91`](https://github.com/gnolang/gno/blob/0b5487b/gnovm/stdlibs/chain/params/params.go#L53) · [↗](../../../../../.worktrees/gno-review-5698/gnovm/stdlibs/chain/params/params.go#L53) — six near-identical wrappers differing only in the typed pointer and the `GetXxx` method name.
