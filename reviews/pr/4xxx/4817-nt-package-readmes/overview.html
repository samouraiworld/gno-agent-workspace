The p/nt/ tree holds gno.land's foundational pure packages: AVL trees, string formatting, DAO primitives, test helpers. Most shipped with no README or a thin auto-generated stub. This work gives every package a proper, hand-written README so a reader can tell what it does and how to use it without reading the source.

The first pass added a generated draft to each package. This branch (docs/nt-readmes) is the promised retouch: it rewrites 20 of them into cleaner, consistent docs, with a descriptive title, a tight usage example, and cross-links between related packages. One new package, bptree, gets its first README here.

Concepts

p/nt/
nt is NewTendermint, the team behind gno.land. The namespace groups the core pure packages the standard library and examples build on.
Pure package (p/)
A p/ package is importable library code with no persisted realm state, unlike a realm (r/). Anyone can import it into a realm.
/v0 and the "Unaudited" banner
Each README opens with a v0 - Unaudited notice: an initial version not yet formally audited, with a fully audited version to follow in a later release. The version suffix comes from an earlier rename of the whole tree to */v0.

What the rewrite changes

Every touched README moves from the generated draft toward the same shape:

  • Descriptive H1. # AVL Tree Package becomes # `avl` - Gas-efficient AVL tree: the package name plus a one-line purpose.
  • Lead sentence that says why it exists, not just what it is. AVL's now names the O(log n) per-op storage cost that motivates it.
  • Tighter usage example that compiles in spirit: shorter comments, idiomatic names (raw, ok := tree.Get(...)), and a realistic second operation (iteration, not just get/set).
  • Cross-links between related packages: seqid points at cford32, pausable at ownable, the commondao extensions back at their parent.
  • Consistent draft banner at the top of each file.

# AVL Tree Package

The `avl` package provides a gas-efficient
AVL tree implementation for storing
key-value data in Gno realms.

## Basic Usage

// This AVL tree will be persisted after
// transaction calls
var tree *avl.Tree

func Get(key string) int {
    // rawValue needs to be converted into
    // the proper type before returning it
    rawValue, exists := tree.Get(key)
    if !exists {
        panic("value at given key does not exist")

After (hand-written)
# `avl` - Gas-efficient AVL tree

A self-balancing AVL tree for storing
key-value data in Gno realms. Each node is
persisted as a separate object, so operations
only load `O(log n)` nodes from storage
instead of the entire collection.

## Usage

// Persisted across transactions.
var tree avl.Tree

func Get(key string) int {
    raw, ok := tree.Get(key)
    if !ok {
        panic("not found")
    }
    return raw.(int)
}

func List() {
    tree.Iterate("", "", func(k string, v any) bool {
        return false // return true to stop early
    })
}
