# gno-onboarding-bot reviews

PR reviews for [samouraiworld/gno-onboarding-bot](https://github.com/samouraiworld/gno-onboarding-bot), the Discord bot that runs the Gno testnet validator onboarding flow (candidate roles, on-chain valoper checks, Google Sheet tracker, reviewer actions).

One directory per PR: `<number>-<short-slug>/` with `review_<model>.md` and the `comment_<model>.md` GitHub draft.
